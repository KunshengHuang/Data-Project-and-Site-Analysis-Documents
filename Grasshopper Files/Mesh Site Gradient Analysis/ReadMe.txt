- 在grasshopper里，mesh并不是由面组成的，而是由 “点、点与点之间的关系” 组成的，所以给Mesh上色时并不是在给面上色，而是在给点上色，这一点请特别注意
- 先选取要进行“坡度分析/可建设用地分析”的Mesh
- 通过电池“Deconstruct Mesh”来获得Mesh在每个点位置的法向量Normal
- 通过电池“Angle”来获得这些法向量和Unit Z向量的夹角α
- 将夹角α输入电池“Tangent”来获得坡度值（无量纲量，即没有单位）
- 通过电池“expression”来写一个嵌套的if（condition，True，False）语句，以此将坡度值分类为四类：
  - 适宜建设用地：0-0.15
  - 可建设用地：0.15-0.25
  - 谨慎建设用地：0.25-0.4
  - 不可建设用地：＞0.4 
- 之后通过电池“Pick n choose”来获得对应坡度分组的一系列颜色
- 将这一系列颜色和Mesh输入电池“Mesh Colors“，即可获得分析后的Mesh
- Bake该Mesh到Rhino中的对应图层
